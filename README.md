# Akgün Erzincan Tulum Peyniri - UI Geliştirilmiş Sürüm

Bu paket mevcut tasarım dili korunarak geliştirildi.

## Eklenenler
- Öne çıkan ürünler vitrini
- Kategori chip filtreleri
- Ürün arama alanı
- Gramaj filtresi
- Stokta olanları göster filtresi
- Ürün sayacı
- Ürün kartlarında kategori, gramaj, puan, stok, etiket ve indirim bilgisi
- Ürün detay modalı
- Sepette ücretsiz kargo ilerleme çubuğu
- WhatsApp sipariş metni oluşturma
- Sepetten ürün silme

## Çalıştırma
Klasörü açıp `index.html` dosyasını tarayıcıda açman yeterli.

## Düzenleme
Ürün bilgileri `script.js` dosyasındaki `products` listesinde tutulur.
